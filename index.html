Plugin objective

  • Change the Html Elements with BsHtml Helper

  • Bake the view templates
  • Flashs Elements

Installation

  • Please read the readme on how to install the plugin with composer (follow link bellow)
  • Then you need to add Html->head() ?> in your layout head tag.
    By default, this add the css and js for bootstrap in your layout

Readme

BsHtml

Add those following lines in your src/Controller/AppController.php file : 

class AppController extends Controller {
public $helpers = [
	...,
	'Html' => [
		'className' => 'Bootstrap.BsHtml'
	],
	'Paginator' => [
		'templates' => 'Bootstrap.bs_paginator.php'
	],
	...
];
}

Alerts

EXAMPLE

Results Code
This is an alert, "success" by default
 echo $this->Html->alert('This is an alert, "success" by default'))
This is an alert with option danger
 echo $this->Html->alert('This is an alert with option danger'), ['type' => 'danger'])
Warning! This is an alert with option warning and dismissable.
echo $this->Html->alert('This is an alert with option warning and dismissable'), ['type' => 'warning', 'dismissable' => true])

Links

Link

Great feature with the links, if a link maps to the url where you are, it adds a 'class' => 'activation' for the link
Results Code
echo $this->Html->link('My Link'), [''])
echo $this->Html->link('My Link with active'), [''])

Default & Tree

Results Code
echo $this->Html->links('default', ['id' => 1]);
echo $this->Html->links('tree', ['id' => 1])

Actif & Principal




echo $this->Html->linksActives(true, 'my-id')

echo $this->Html->linksActives(false, 'my-id', [
"active" => [
"label" => "danger",
"name" => "Desactivé",
"btn_name" => __d('bootstrap', "Activer"),
"btn_icon" => 'check',
'action' => 'desactivate',
'class' => 'btn btn-danger btn-lg'
]
])
					
echo $this->Html->linksPrincipal(true, 'my-id')

echo $this->Html->linksPrincipal(false, 'my-id', [
	'principal' => [
		'name' => 'In principal'
	],
	'not_principal' => [
		'name' => 'Not in Principal'
	]
])

BsForm

Add those following lines in your src/Controller/AppController.php file : 

class AppController extends Controller {
public $helpers = [
		...,
		'Form' => [
			'templates' => 'Bootstrap.bs_form.php',
			'className' => 'Bootstrap.BsForm',
		],
		...
	];
}

Bake

Bake Bootstrap Themes

You can launch the cake bake view with templates provided by this plugin and made for Bootstrap
When you bake your views with cake bake, just add --theme bootstrap, like this: 

./bin/cake bake view Users --theme bootstrap

Add Flash Elements

You can add flash elements in your layouts: 

element('Bootstrap.Flash/flashs') ?>
you'll be then able to render error, warning, danger and success flash messages in your controllers. 
Flash->set(__('this could be saved.'), ['key' => 'success']) ?>
Flash->set(__('Pb this could not be saved. Try again.'), ['key' => 'error']) ?>
Or/And add Auth flash elements 
element('Bootstrap.Flash/auth') ?>
