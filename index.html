To easily use Bootstrap Twitter 3.x in CakePHP projects

With this plugin, you can do the following things:
- Change the Html Elements with BsHtml Helper - Change the Form Elements with BsForm Helper - Bake the view templates - Example of layout

Alerts

EXAMPLE

Results Code
This is an alert, "success" by default
 echo $this->Html->alert('This is an alert, "success" by default'))
This is an alert with option danger
 echo $this->Html->alert('This is an alert with option danger'), ['type' => 'danger'])
Warning! This is an alert with option warning and dismissable.
echo $this->Html->alert('This is an alert with option warning and dismissable'), ['type' => 'warning', 'dismissable' => true])

Images & Icones

EXAMPLE

Results Code
echo $this->Html->image('PluginName.logo.png')
echo $this->Html->icon('euro');
echo $this->Html->icon('search');

Buttons, Labels & Badges

Buttons

Results Code
echo $this->Html->button('This is a button with default'))
echo $this->Html->button('This is a button with option primary and a class and id defined'), ['type' => 'primary', 'class' => 'btn-xs'])
echo $this->Html->button('This is a button with option danger and a class and id defined'), ['type' => 'danger', 'class' => 'btn-md my-class', 'id' => 'my-id'])

Labels

Results Code
This is a label with default echo $this->Html->label('This is a label with default')

Badges

Results Code
This is a badge with default echo $this->Html->badge('This is a badge with default');

Links

Link

Results Code
echo $this->Html->link('My Link'), [''])
echo $this->Html->link('My Link with active'), [''])

Default & Tree

Results Code
echo $this->Html->links('default', ['id' => 1]);
echo $this->Html->links('tree', ['id' => 1])

Actif & Principal

Results Code


echo $this->Html->links_actif(true, 'my-id')

echo $this->Html->links_principal(true, 'my-id')
